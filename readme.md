# Fiftieth
> The website for the 50th Golden Anniversary

## Contribution

### Requirements

- Node / npm / bower
- Ruby / RubyGems

### Setup

Just dish this script and you should be good to go!

```
gem install
open index.html
```

<sub>Note: For **Windows** users, manually open `index.html` if the second line doesn't work for you.</sub>
